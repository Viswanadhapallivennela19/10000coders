Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and sScripting languages such asJavaScript. Web browsers receive HTML documents from a web server or from local storage and render documents into multimedia web pages. HTML describes the structure of web page semantically and originally included cues for its appearance.
